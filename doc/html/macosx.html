Download VICE for Mac OS X systems

You can download the latest VICE distributions for Mac OS X systems here. Some distributions may not be available at the release date, they will be added as they become available.

Source

  • Tarball: vice-3.1.tar.gz

SDL UI

  • Intel Mac (64 Bit)
  • 64bit Binary for Intel Mac OS X >= 10.12: vice-macosx-sdl-x86_64-10.12-3.1.dmg

Cocoa UI

  • Intel Mac (64 Bit)
  • 64bit Binary for Intel Mac OS X >= 10.12: vice-macosx-cocoa-x86_64-10.12-3.1.dmg
    • please be aware that the Cocoa UI is lagging behind compared to the SDL version, it is recommended to use the SDL download.

Old (VICE 3.0) Cocoa UI

  • Power PC or Intel Mac (32 / 64 Bit)
  • 64bit Binary for Intel Mac OS X >= 10.12: vice-macosx-cocoa-x86_64-10.12-3.0.dmg

Old (VICE 3.0) SDL UI

  • Power PC or Intel Mac (32 / 64 Bit)
  • 64bit Binary for Intel Mac OS X >= 10.12: vice-macosx-sdl-x86_64-10.12-3.0.dmg

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
