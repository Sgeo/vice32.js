Download VICE for Darwin systems

You can download the latest VICE distributions for Darwin systems here. Some distributions may not be available at the release date, they will be added as they become available.

Source distributions

  • Tarball: vice-3.1.tar.gz

Binary distributions

  • Binary for Intel Darwin systems: VICE-3.1-darwin-i386.tar.bz2. (not available yet) Unpack in /usr/local
  • Binary for PureDarwin systems: VICE-3.1-puredarwin-i386.tar.bz2. (not available yet) Unpack in /usr/local
  • Binary for PPC Darwin systems: VICE-3.1-darwin-ppc.tar.bz2. (not available yet) Unpack in /usr/local

Old (VICE 3.0) binary distributions

  • Binary for Intel Darwin systems: VICE-3.0-darwin-i386.tar.bz2. Unpack in /usr/local
  • Binary for PureDarwin systems: VICE-3.0-puredarwin-i386.tar.bz2. (not available yet) Unpack in /usr/local
  • Binary for PPC Darwin systems: VICE-3.0-darwin-ppc.tar.bz2. Unpack in /usr/local

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
