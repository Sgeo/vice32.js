Download VICE for BeOS systems

You can download the latest VICE distributions for BeOS systems here. Some distributions may not be available at the release date, they will be added as they become available.

Source distributions

  • Tarball: vice-3.1.tar.gz

Binary distributions

  • Binary for Intel BeOS/Zeta systems: BeVICE-3.1.x86.zip.
  • Binary for PowerPC BeOS systems: BeVICE-3.1.powerpc.zip.
  • Binary for 32bit Haiku systems: BeVICE-3.1.Haiku.zip.
  • Binary for 64bit Haiku systems: BeVICE-3.1.Haiku64.zip.

Old (VICE 3.0) binary distributions

  • Binary for Intel BeOS/Zeta systems: BeVICE-3.0.x86.zip.
  • Binary for PowerPC BeOS systems: BeVICE-3.0.powerpc.zip.
  • Binary for 32bit Haiku systems: BeVICE-3.0.Haiku.zip.
  • Binary for 64bit Haiku systems: BeVICE-3.0.Haiku64.zip.

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.

Old binary distributions

Binaries of old releases can be found on http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
