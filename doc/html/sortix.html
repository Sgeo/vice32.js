Download VICE for Sortix systems

You can download the latest VICE distributions for Sortix systems here. Some distributions may not be available at the release date, they will be added as they become available.

Source distributions

  • Tarball: vice-3.1.tar.gz

Binary distributions

  • Binary for 32bit Sortix systems: SDLVICE-3.1-Sortix32.tix.tar.xz. (Experimental) (Not available yet)
  • Binary for 64bit Sortix systems: SDLVICE-3.1-Sortix64.tix.tar.xz. (Experimental) (Not available yet)

Old (VICE 3.0) binary distributions

  • Binary for 32bit Sortix systems: SDLVICE-3.0-Sortix32.tix.tar.xz. (Experimental)
  • Binary for 64bit Sortix systems: SDLVICE-3.0-Sortix64.tix.tar.xz. (Experimental)

The files are taken from http://www.zimmers.net/anonftp/pub/cbm/crossplatform/emulators/VICE.
